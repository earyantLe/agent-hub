快速开始

# 克隆仓库
git clone https://github.com/earyant/all_app_as_agent.git
cd all_app_as_agent

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 运行测试
pnpm test

核心 API

GET /api/skills

获取技能列表，支持搜索和分页

curl http://localhost:3000/api/skills?q=search&limit=20
GET /api/skills/:name

获取技能详情

curl http://localhost:3000/api/skills/demo-skill
POST /api/skills

提交新技能

curl -X POST http://localhost:3000/api/skills \
  -H "Content-Type: application/json" \
  -d '{"descriptor": {...}}'
PUT /api/skills/:id

更新技能信息或审核状态

DELETE /api/skills/:id

删除技能

POST /api/skills/batch/delete

批量删除技能

POST /api/skills/batch/approve

批量审核技能

GET /health

健康检查
