什么是 Agent Hub？

Agent Hub 是一个 Agent 技能市场，帮助你将任何网站/API 转换为 AI Agent 可调用的技能。

  • 发现 Agent - 浏览已注册的 Agent 技能，直接集成到你的 AI 应用中
  • 转换网站 - 输入任意网站 URL，自动生成 Skill Descriptor
  • 发布技能 - 将你的 API 或服务发布到市场，供他人使用
