# Agent Hub 架构设计

## 系统概览

```
┌─────────────────────────────────────────────────────────────┐
│                    Agent Hub Ecosystem                       │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────────┐     ┌──────────────┐     ┌──────────────┐ │
│  │   Registry   │◄────│   Protocol   │────►│     SDK      │ │
│  │   (市场)      │     │   (协议层)    │     │   (客户端)    │ │
│  └──────────────┘     └──────────────┘     └──────────────┘ │
│                            ▲                                 │
│                            │                                 │
│                     ┌──────┴──────┐                          │
│                     │ Transformer │                          │
│                     │  (转换工具)  │                          │
│                     └─────────────┘                          │
└─────────────────────────────────────────────────────────────┘
```

## 核心组件

### Protocol (协议层)

- Skill Descriptor JSON Schema
- OpenAPI 规范定义
- 验证工具

### Registry (市场服务)

- REST API (Fastify)
- PostgreSQL 数据库
- 管理 UI (Next.js)

### Transformer (转换工具)

- HTML 解析器 (Cheerio)
- 浏览器自动化 (Playwright)
- Skill 生成器

### SDK (客户端)

- TypeScript SDK
- Python SDK
